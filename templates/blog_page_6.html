설명 가능한 AI의 핵심 기술들

XAI는 AI 모델의 의사결정 과정을 이해할 수 있게 해주는 중요한 기술입니다

안녕하세요! 오늘은 XAI(Explainable AI)에 대해 자세히 알아보겠습니다. AI 모델의 의사결정 과정을 이해하고 해석할 수 있게 해주는 핵심 기술입니다.

XAI의 기본 개념

XAI란: AI 모델의 의사결정 과정을 인간이 이해할 수 있도록 설명하는 기술입니다. 이를 통해 AI의 투명성과 신뢰성을 확보할 수 있습니다.

Grad-CAM (Gradient-weighted Class Activation Mapping)

정의: CNN의 특징맵(feature map)과 클래스 점수의 gradient를 활용해 시각적 중요 영역을 강조합니다.

특징: 기존 학습된 모델 파라미터를 수정하지 않고 사후적(post-hoc) 시각화가 가능하며, ReLU를 적용해 양의 기여(positive contribution)만 강조합니다.

Grad-CAM을 통한 이미지 분류 설명

📝 Grad-CAM 특징
  • 클래스 점수에 대한 특징맵 채널별 기울기를 이용해 중요도를 계산
  • ReLU를 적용하여 예측에 긍정적 영향을 미친 영역만 강조
  • 기존 학습된 CNN 모델에 바로 적용 가능
  • 주로 비전(이미지) 모델 설명에 활용

LIME (Local Interpretable Model-agnostic Explanations)

정의: 복잡한 모델의 예측을 부분적으로 단순한 대리모델(surrogate model)로 근사합니다.

특징: 특정 샘플 주변에 데이터를 무작위로 생성(perturbation)하고, 원 모델 예측값을 학습 데이터로 사용합니다. Model-agnostic하며 전역보다는 국소(local) 설명에 특화됩니다.

LIME의 로컬 설명 과정

SHAP (SHapley Additive exPlanations)

정의: 게임 이론의 Shapley Value 기반으로 각 피처의 기여도를 정량화합니다.

특징: 모든 피처 조합을 고려하므로 이론적으로 공정(consistency)하나, 계산량이 큽니다. 모델 특화 최적화(TreeSHAP 등)로 계산 효율성을 확보할 수 있습니다.

SHAP을 통한 피처 중요도 분석

CAM (Class Activation Map)

정의: 특정 클래스에 대한 예측을 위해 Global Average Pooling(GAP) 뒤의 fully connected layer 가중치를 이용해 feature map 가중합(weighted sum)을 만듭니다.

특징: GAP 계층이 필요하여 모델 아키텍처 변경이 필요하며, CNN의 마지막 합성곱 계층과 GAP을 통해 클래스별 "어디를 보고 판단했는가?" 시각화합니다.

CAM을 통한 클래스별 활성화 영역

XAI의 중요성

  • 투명성 확보: AI 모델의 의사결정 과정을 이해할 수 있음
  • 신뢰성 향상: 사용자가 AI의 판단을 신뢰할 수 있음
  • 규제 준수: AI 규제 및 윤리 가이드라인 준수
  • 디버깅 용이: 모델 오류의 원인을 파악할 수 있음
