AI 모델의 정규화 완벽 가이드

머신러닝과 딥러닝에서 중요한 데이터 전처리 기법

정규화는 AI 모델의 성능 향상에 중요한 역할을 합니다

안녕하세요! 오늘은 AI 모델에서 매우 중요한 개념인 정규화(Normalization)에 대해 자세히 알아보겠습니다. 정규화는 데이터의 스케일을 조정하여 특정 범위로 맞추거나, 벡터의 노름을 1로 만드는 과정입니다.

정규화의 핵심 개념

정규화의 핵심: 결국 각 요소를 벡터의 Norm으로 나누는 것입니다. 이를 통해 데이터의 스케일을 일정하게 맞추고, 모델의 학습 안정성을 높일 수 있습니다.

L1 Normalization (정규화)

정의: 벡터의 각 요소를 벡터의 L1 Norm으로 나눕니다. 결과적으로 벡터의 L1 Norm은 1이 됩니다.

수식: x̂ = x / ||x||₁

특징: 벡터의 모든 요소의 절댓값 합이 1이 되도록 만듭니다. 데이터의 비율에 중점을 둘 때 유용합니다.

L1 정규화의 개념을 시각적으로 표현

📝 L1 정규화 예시

벡터 x = [3, -4, 5]일 때:

  • L1 Norm = |3| + |-4| + |5| = 12
  • L1 정규화된 벡터 x̂ = [3/12, -4/12, 5/12] = [0.25, -0.333, 0.417]
  • x̂의 L1 Norm = |0.25| + |-0.333| + |0.417| ≈ 1

L2 Normalization (정규화)

정의: 벡터의 각 요소를 벡터의 L2 Norm으로 나눕니다. 결과적으로 벡터의 L2 Norm은 1이 됩니다. 이를 단위 벡터(Unit Vector)라고도 합니다.

수식: x̂ = x / ||x||₂

특징: 벡터의 방향을 유지하면서 크기를 1로 만듭니다. 주로 코사인 유사도(Cosine Similarity) 계산 등 벡터 간의 방향성 비교에 사용됩니다.

L2 정규화는 벡터의 방향을 유지하면서 크기를 1로 만듭니다

📝 L2 정규화 예시

벡터 x = [3, -4, 5]일 때:

  • L2 Norm = √(3² + (-4)² + 5²) = √(9 + 16 + 25) = √50 ≈ 7.07
  • L2 정규화된 벡터 x̂ = [3/√50, -4/√50, 5/√50] ≈ [0.424, -0.566, 0.707]
  • x̂의 L2 Norm = √(0.424² + (-0.566)² + 0.707²) ≈ √(0.180 + 0.320 + 0.500) = √1 = 1

L1과 L2 정규화의 차이점 비교

실제 적용 예시

Python으로 정규화 구현하기:

import numpy as np

def l1_normalize(x):
    return x / np.sum(np.abs(x))

def l2_normalize(x):
    return x / np.linalg.norm(x)

# 예시 벡터
x = np.array([3, -4, 5])

# L1 정규화
x_l1 = l1_normalize(x)
print("L1 정규화:", x_l1)

# L2 정규화  
x_l2 = l2_normalize(x)
print("L2 정규화:", x_l2)

실제 코드로 정규화를 구현한 결과

정규화의 장점

  • 학습 안정성: 특성 간 스케일 차이로 인한 학습 불안정성 해결
  • 수렴 속도: 그래디언트 디센트의 수렴 속도 향상
  • 모델 성능: 전반적인 모델 성능 향상
  • 해석 가능성: 정규화된 특성은 해석하기 쉬움

정규화 적용 전후의 모델 성능 비교

정규화 vs 표준화

구분 정규화 (Normalization) 표준화 (Standardization)
범위 [0, 1] 또는 [-1, 1] (-∞, +∞)
평균 변화함 0
표준편차 변화함 1
이상치 영향 민감함 상대적으로 덜 민감함

정규화의 활용 분야

  • 이미지 처리: 픽셀 값을 [0, 1] 범위로 정규화
  • 자연어 처리: 단어 임베딩 벡터의 정규화
  • 추천 시스템: 사용자/아이템 벡터의 정규화
  • 신경망: 활성화 함수 입력의 정규화
