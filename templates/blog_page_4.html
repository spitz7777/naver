그래프 증강 기법 완벽 가이드

그래프 신경망에서 데이터 다양성을 확보하는 방법

그래프 증강은 그래프 신경망의 성능 향상에 중요한 역할을 합니다

안녕하세요! 오늘은 그래프 증강(Graph Augmentation)에 대해 자세히 알아보겠습니다. 그래프 신경망(GNN)에서 데이터 다양성을 확보하고 모델 성능을 향상시키는 핵심 기술입니다.

그래프 증강의 기본 개념

그래프 증강이란: 기존 그래프의 구조나 속성을 변형하여 새로운 그래프 샘플을 생성하는 기법입니다. 이를 통해 그래프 신경망의 일반화 성능을 향상시킬 수 있습니다.

Graph Diffusion (그래프 확산)

정의: Personalized PageRank(PPR), Heat Kernel 등을 활용해 그래프의 연결 가중치를 재계산합니다.

특징: 장거리 의존성(Long-range dependency)을 강화하고 정보 전달을 원활히 하며, 노드 수를 늘리는 것이 아니라 가중치만 변형합니다.

Graph Diffusion을 통한 가중치 재계산

Graph Mixup (그래프 혼합)

정의: 두 그래프의 특징(Feature)과 레이블(Label)을 선형 결합하여 새로운 학습 샘플을 생성합니다.

특징: 그래프 분류(Graph Classification) 문제에서 데이터 다양성을 확보하며, 결정 경계를 부드럽게 만들어 일반화 성능을 높입니다.

Graph Mixup을 통한 그래프 합성

Node Attribute Shuffling (속성 셔플링)

정의: 노드 특성(Feature)을 무작위로 섞어 속성과 구조의 결합을 교란합니다.

특징: Contrastive Learning에서 "속성 무시 뷰(Attribute-agnostic view)" 생성을 위해 활용되며, 모델이 구조 정보와 속성 정보 모두를 균형 있게 학습하도록 합니다.

Node Attribute Shuffling의 개념

Edge Dropout / DropEdge

정의: 일부 엣지를 무작위로 제거하여 과적합과 오버스무딩(oversmoothing)을 방지합니다.

특징: 그래프의 연결을 희소하게 만들어 학습 안정성을 높이며, Contrastive Learning에서 뷰 생성용 증강 기법으로 사용됩니다.

Edge Dropout을 통한 엣지 제거

그래프 증강의 장점

  • 구조적 다양성: 다양한 그래프 구조로 모델 학습
  • 과적합 방지: 구조적 변형으로 일반화 성능 향상
  • Robustness 강화: 불완전한 그래프에 대한 내성 향상
  • Contrastive Learning: 자기지도 학습에서 효과적
